# Plan an academic week

Create a realistic plan for classes, assignments, study, and project work

Version 0.1

## Priority framework

- Deadline risk
- Academic importance
- Dependency impact
- Estimated effort
- Consequence of delay

## Daily kickoff

- Open the required source
- Read the task definition
- Start the named first action
- Set the planned stop point
- Record status before switching

## Missed-task recovery

Choose one: reschedule with a buffer; reduce scope within requirements; ask for clarification; renegotiate when appropriate; remove obsolete work. Never silently overload the next day.

## Weekly plan quality review

- Every deadline traces to an official source
- Large assignments have concrete next actions
- Dependencies are scheduled first
- Planned work fits available capacity
- Buffers and recovery time are present
- Private information is excluded from unapproved tools

## Risk notes

- Review AI-assisted output for accuracy before use.
- Do not include private health, disability, or account information in unapproved AI tools.
